Pensamento computacional é a habilidade de mobilizar os conhecimentos para solucionar problemas com eficiência, a partir do uso da tecnologia.

Nós fazemos parte do Programa Edutech, que é um programa que tem como objetivo ensinar os estudantes uma habilidade em programação.Trazendo a oportunidade de crescimento intelectual e possibilidade de abertura para o mercado de trabalho.
